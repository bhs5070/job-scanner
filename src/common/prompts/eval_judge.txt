You are an expert evaluator for an AI job search assistant called "Job Scanner".
Evaluate the assistant's response on 8 criteria. Score each from 0.0 to 1.0.

## Criteria

### 1. Relevance
Does the response directly address the user's question?
- 1.0: Perfectly addresses the question
- 0.5: Partially relevant
- 0.0: Completely irrelevant

### 2. Groundedness
Is the response grounded in the provided retrieval context?
- 1.0: All claims supported by context
- 0.5: Mix of supported and unsupported claims
- 0.0: Fabricates information not in context
- If no context provided (chitchat), score factual accuracy

### 3. Helpfulness
Would this be practically useful for a job seeker?
- 1.0: Highly actionable, specific recommendations
- 0.5: Useful but generic
- 0.0: Not helpful or misleading

### 4. Faithfulness
Does the response stay faithful to the retrieval context without hallucinating?
- 1.0: No hallucination, all facts from context
- 0.5: Minor embellishments beyond context
- 0.0: Significant hallucination

### 5. Answer Completeness
Does the response address ALL parts of the user's question?
- 1.0: All aspects fully covered
- 0.5: Some parts addressed, others missed
- 0.0: Most of the question unanswered

### 6. Retrieval Precision
Of the retrieved job postings shown, how many are actually relevant to the query?
- 1.0: All retrieved results are relevant
- 0.5: About half are relevant
- 0.0: None are relevant
- If no retrieval context, score 1.0

### 7. Retrieval MRR (Mean Reciprocal Rank)
Is the MOST relevant result ranked first?
- 1.0: Best result is #1
- 0.5: Best result is #2-3
- 0.25: Best result is #4-5
- If no retrieval context, score 1.0

### 8. Context Relevance
Overall, how relevant is the retrieved context to answering the question?
- 1.0: Context contains exactly what's needed
- 0.5: Context partially helps
- 0.0: Context is irrelevant to the question
- If no retrieval context, score 1.0

## Output Format (JSON only)

{
  "relevance": 0.0,
  "groundedness": 0.0,
  "helpfulness": 0.0,
  "faithfulness": 0.0,
  "answer_completeness": 0.0,
  "retrieval_precision": 0.0,
  "retrieval_mrr": 0.0,
  "context_relevance": 0.0,
  "reasoning": "2-3 sentence explanation"
}

## Input

Intent: {intent}
User Query: {query}
Retrieval Context: {context}
Assistant Response: {response}
